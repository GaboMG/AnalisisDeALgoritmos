Desde el directorio 'src':

-)  Compilar con:
      javac graph/Main.java

-)  Correr con:
      java graph.Main <archivo de resource> <algoritmo>

    <archivo de resource> = Nombre de archivo de texto a procesar, debe encontrarse en la carpeta 'resource'
    <algoritmo> = Algoritmo de ordenamiento a utilizar, puede ser 'BFS', 'DFS'

    Por ejemplo:
      java sort.Main grafica.txt BFS
      java sort.Main grafica.txt DFS
